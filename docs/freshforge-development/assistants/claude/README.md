## How to Use with Claude

2. Paste **`INSTRUCTIONS.md`** into custom instructions.
3. Upload `CURRENT-STATE.md`, `PROJECT-OVERVIEW.md`, `ARCHITECTURE-AND-CODE.md`.
4. After managed phases in this repo, re-upload at least `CURRENT-STATE.md`.
